01

Vision

Why this exists

Type what you want to say. AI turns it into street art. It appears on the wall in AR for anyone nearby to discover. The city becomes a living, evolving canvas — every surface a potential gallery, every passerby a potential artist.

Core Pitch
🎨

Democratized Street Art

No spray cans, no risk, no permits. Anyone with a phone becomes a muralist. The AI translates intent into gritty, textured graffiti that belongs on a brick wall.

📡

GPS-Anchored Echoes

Each piece is pinned to exact coordinates. Walk past a bridge and discover what someone tagged there last night. Art that haunts specific places, not feeds.

👁️

Invisible Layer

The city gains a secret dimension visible only through your camera. A parallel world of expression layered on top of concrete and glass.

04

User Journey

From stranger to street artist

Step 01 — Arrive

Land on the Surface

User opens the app on their phone. The landing page loads instantly — dark, electric, inviting. Two clear paths: explore what's already here, or go straight to AR. No signup. No friction. Just in.

Next.js SSR Tailwind
Step 02 — Discover

See the Invisible Layer

Through the map or AR camera, nearby graffiti materializes. Each piece floats at the exact spot someone stood when they created it. The user walks through a gallery that only exists through their screen — art tethered to bridges, walls, benches, and doorways.

Leaflet.js A-Frame AR.js Geolocation API
Step 03 — Create

Leave Your Mark

Tap the floating action button. Type a message, a thought, a declaration. The AI engine transforms raw text into spray-painted graffiti art — textured, layered, dripping with character. It materializes at your feet in seconds.

Gemini API Supabase Storage GPS Coordinates
Step 04 — Echo

Your Art Echoes Forward

Your creation is now pinned to that exact spot. Tomorrow, next week, next month — anyone who walks past with the app open will discover what you left there. Your words, transformed into art, echoing through the city long after you've gone.

PostgreSQL Spatial Query Public API

Data Schema

The shape of an echo

supabase/schema.sql 
CREATE TABLE graffiti (
  id          UUID          PRIMARY KEY DEFAULT gen_random_uuid(),
  prompt      TEXT          NOT NULL,          -- what the user typed
  image_url   TEXT          NOT NULL,          -- Supabase Storage URL
  latitude    FLOAT8        NOT NULL,          -- GPS lat
  longitude   FLOAT8        NOT NULL,          -- GPS lng
  creator     TEXT          DEFAULT 'Anonymous',  -- optional nickname
  created_at  TIMESTAMPTZ   DEFAULT now()        -- when it was born
);

-- RLS: public read + insert, no auth required
ALTER TABLE graffiti ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Anyone can read"
  ON graffiti FOR SELECT USING (true);

CREATE POLICY "Anyone can create"
  ON graffiti FOR INSERT WITH CHECK (true);

06

Resilience

When the wall fights back

GPS Denied

Graceful fallback banner explaining that location is required. Map defaults to a demo area. AR shows instructional overlay rather than blank camera.

Camera Blocked

AR page shows stylized placeholder with instructions. Redirect to map view as the primary discovery mode. No dead ends.

AI Generation Fail

Toast notification with retry button. If API is fully down, show a curated fallback image with the user's text overlaid in graffiti font.

Offline / Slow

Service worker caches the shell. Previously loaded graffiti persists in local state. Creation queued for when connectivity returns.

No Nearby Art

Instead of empty state, show encouraging message: "Be the first to tag this spot." Pull in the nearest pieces as distant pins on the map edge.

AR Drift

GPS accuracy varies indoors. Show accuracy indicator. If accuracy > 100m, warn user and suggest moving outdoors for best experience.

07

Demo Strategy

The 3-minute pitch

1 Pre-seed venue with 5-10 pieces
2 Show landing page on projector
3 Open map view — show pins
4 Switch to AR — walk toward art
5 Live create a new piece on stage

Backup plan: if GPS is unreliable indoors, pre-record a 30-second AR walkthrough video outside the venue and play it during the pitch. Always have the live map as a reliable fallback.
